## Get Credentials

<CodeGroup>
```bash GET
curl https://your-domain.com/api/get-credentials.php
```
</CodeGroup>

Retrieves all API credentials with sensitive values masked.

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if credentials were retrieved successfully
</ResponseField>

<ResponseField name="whatsapp" type="object">
  WhatsApp Business API credentials
  
  <ResponseField name="phone_number_id" type="string">
    WhatsApp phone number ID (visible)
  </ResponseField>
  
  <ResponseField name="access_token" type="string">
    API access token (masked as `••••••••`)
  </ResponseField>
  
  <ResponseField name="app_secret" type="string">
    App secret for webhook validation (masked)
  </ResponseField>
  
  <ResponseField name="verify_token" type="string">
    Webhook verification token (visible)
  </ResponseField>
  
  <ResponseField name="has_credentials" type="boolean">
    Whether complete WhatsApp credentials are configured
  </ResponseField>
</ResponseField>

<ResponseField name="openai" type="object">
  OpenAI API credentials
  
  <ResponseField name="api_key" type="string">
    OpenAI API key (masked as `••••••••`)
  </ResponseField>
  
  <ResponseField name="model" type="string">
    GPT model name (e.g., `"gpt-4o"`, `"gpt-3.5-turbo"`)
  </ResponseField>
  
  <ResponseField name="embedding_model" type="string">
    Embedding model for RAG (e.g., `"text-embedding-3-small"`)
  </ResponseField>
  
  <ResponseField name="has_credentials" type="boolean">
    Whether OpenAI credentials are configured
  </ResponseField>
</ResponseField>

<ResponseField name="google" type="object">
  Google OAuth credentials for Calendar API
  
  <ResponseField name="client_id" type="string">
    Google OAuth client ID (visible)
  </ResponseField>
  
  <ResponseField name="client_secret" type="string">
    OAuth client secret (masked)
  </ResponseField>
  
  <ResponseField name="access_token" type="string">
    OAuth access token (masked)
  </ResponseField>
  
  <ResponseField name="refresh_token" type="string">
    OAuth refresh token (masked)
  </ResponseField>
  
  <ResponseField name="calendar_id" type="string">
    Google Calendar ID (visible)
  </ResponseField>
  
  <ResponseField name="has_credentials" type="boolean">
    Whether complete Google credentials are configured
  </ResponseField>
</ResponseField>

## Save Credentials

<CodeGroup>
```bash POST
curl -X POST https://your-domain.com/api/save-credentials.php \
  -H "Content-Type: application/json" \
  -d '{
    "service": "openai",
    "api_key": "sk-proj-...",
    "model": "gpt-4o",
    "embedding_model": "text-embedding-3-small"
  }'
```
</CodeGroup>

Saves encrypted credentials for a specific service.

### Request

<ParamField body="service" type="string" required>
  Service to configure: `"whatsapp"`, `"openai"`, or `"google"`
</ParamField>

#### WhatsApp Parameters

<ParamField body="phone_number_id" type="string">
  WhatsApp Business phone number ID
</ParamField>

<ParamField body="access_token" type="string">
  WhatsApp API access token (will be encrypted)
</ParamField>

<ParamField body="app_secret" type="string">
  App secret for webhook validation (will be encrypted)
</ParamField>

<ParamField body="verify_token" type="string">
  Webhook verification token
</ParamField>

#### OpenAI Parameters

<ParamField body="api_key" type="string">
  OpenAI API key (will be encrypted)
</ParamField>

<ParamField body="model" type="string">
  GPT model name (e.g., `"gpt-4o"`)
</ParamField>

<ParamField body="embedding_model" type="string">
  Embedding model for RAG (e.g., `"text-embedding-3-small"`)
</ParamField>

#### Google Parameters

<ParamField body="client_id" type="string">
  Google OAuth client ID
</ParamField>

<ParamField body="client_secret" type="string">
  OAuth client secret (will be encrypted)
</ParamField>

<ParamField body="access_token" type="string">
  OAuth access token (will be encrypted)
</ParamField>

<ParamField body="refresh_token" type="string">
  OAuth refresh token (will be encrypted)
</ParamField>

<ParamField body="calendar_id" type="string">
  Google Calendar ID to use
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if credentials were saved successfully
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message
</ResponseField>

<ResponseField name="error" type="string">
  Error description (only present on failure)
</ResponseField>

### Examples

<CodeGroup>
```json WhatsApp
{
  "service": "whatsapp",
  "phone_number_id": "123456789",
  "access_token": "EAAxxxx...",

```

```json OpenAI

  "api_key": "sk-proj-...",

```
</CodeGroup>

### Success Response

```json
{
  "success": true,
  "message": "Credenciales guardadas exitosamente"
}
```

### Error Responses

```json Invalid Service
{
  "error": "Invalid service. Use: whatsapp, openai, or google"
}
```

```json Missing Service Field
{
  "error": "Invalid input. \"service\" field required."
}
```

## Error Handling

| Status Code | Description |
|-------------|-------------|
| 200 | Credentials saved successfully |
| 400 | Invalid input or service |
| 405 | Method not allowed (GET/POST required) |
| 500 | Internal server error |

## Security

<Warning>
  All sensitive credentials (tokens, secrets, API keys) are encrypted using AES-256-CBC before storage. The encryption key should be configured in your environment.
</Warning>

Source implementation:
- GET: `api/get-credentials.php:12-47`
- POST: `api/save-credentials.php:28-65`
- Encryption: Uses `App\Services\EncryptionService` and `App\Services\CredentialService`
